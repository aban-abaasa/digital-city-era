# Digital City Era

A comprehensive digital supermarket platform designed to streamline retail operations with multi-role access and real-time management.

## What It Does

Digital City Era is a full-stack supermarket management system that connects all key stakeholders:

- **Admins** - Manage users, system configuration, and generate reports
- **Managers** - Oversee branch operations, sales, and inventory
- **Cashiers** - Process transactions with barcode scanning and payment handling
- **Suppliers** - Manage products and stock levels
- **Customers** - Browse and purchase products via mobile-friendly interface

## Key Features

- 🔐 Secure authentication with Google OAuth and email verification
- 💳 Multiple payment methods (cash and card)
- 📱 Fully responsive mobile design
- 📊 Real-time inventory tracking
- 🛒 Shopping cart and order management
- 📦 Barcode scanner integration
- 👥 Role-based access control

## Tech Stack

React frontend | Node.js backend | PostgreSQL database | Google OAuth

## Getting Started

1. Clone: `git clone https://github.com/aban-abaasa/digital-city-era..git`
2. Install: `npm install`
3. Configure `.env` with database and OAuth credentials
4. Run migrations: `npm run db:migrate`
5. Start: `npm start`

## Access the Application

The application runs on **port 7779**:

- **Local**: http://localhost:7779/
- **Network (192.168.137.x)**: http://192.168.137.1:7779/
- **Network (192.168.1.x)**: http://192.168.1.22:7779/

Use any of these URLs to access the platform from your local machine or network devices.
